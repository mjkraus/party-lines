#Party L/nes

Party L/nes is an exercise in interactive storytelling. I was inspired to create this site because of works I have seen online where the user becomes immersed in a story through web design.

##Tools
* Node.js
* JavaScript
* jQuery
* Highcharts.js
* Sparklines.js
* Bootstrap
* CSS
* HTML
